# Hexfall Clone - HexagonYigit

This is a sample demo project made for a job application. No 3rd party library has been used other than what comes pre-installed with Unity.

Here are some tips;
- You can change the **game grid** from **Grid Manager** under **Managers**.
- You can edit the colors from **Color Manager** again under **Managers**.
